# 💰 Cashback API

API developed to calculate cashback and store user query history based on client IP address.

---

## 📌 About the Project

The **Cashback API** allows you to:

- Calculate cashback based on a given amount
- Store each request in the database
- Identify users through their IP address
- Retrieve cashback history by IP

This project was built with a focus on **code organization, separation of concerns, and backend best practices**.

---

## 🚀 Technologies

- Python
- FastAPI
- SQLAlchemy
- PostgreSQL (or SQLite for development)
- Docker (optional)
